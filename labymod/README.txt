README - Lootsucht.eu LabyMod Setup

1. Gehe auf https://github.com und erstelle ein neues Repository (z.B. "labymod").
2. Lade die Dateien aus diesem ZIP hoch.
3. Aktiviere GitHub Pages unter: Repository → Settings → Pages → Branch auswählen (z.B. "main" oder "master").
4. Nach ein paar Sekunden wird dein Setup z.B. unter https://deinname.github.io/labymod/server.json erreichbar sein.
5. Wichtig: Dein Minecraft-Server (z.B. lootsucht.eu) muss zur Domain der JSON passen, oder du nutzt SRV-Records.

Dateien in diesem ZIP:
- server.json (Konfiguration)
- background.jpg (1920x1080 Hintergrund)
- icon.png (256x256 Server-Icon)

Du kannst die Texte oder Bilder jederzeit anpassen.
